# ЭкспортКомпас CRM

Полнофункциональная локальная CRM для ведения базы компаний, экспортёров, контактов Минпрома/ЦПЭ, задач, мер поддержки, истории взаимодействий и аналитики.

## Самый простой запуск без pgAdmin и без PostgreSQL

В этой версии приложение работает **сразу локально**. Внешний PostgreSQL больше не обязателен.

Если файла `.env` нет, приложение автоматически использует встроенную файловую PostgreSQL-совместимую базу **PGlite** и создаёт данные в папке:

```text
.local-db/export-compass
```

### Запуск

```bash
npm install
npm run dev
```

Откройте:

```text
http://localhost:3000
```

При первом открытии приложение автоматически:

1. создаст таблицы;
2. добавит демонстрационные компании;
3. запустит CRM-интерфейс.

## Если хотите полностью сбросить локальную встроенную базу

Остановите сайт `Ctrl + C`, удалите папку:

```text
.local-db
```

Потом снова:

```bash
npm run dev
```

## Если хотите использовать настоящий PostgreSQL через pgAdmin

Это необязательно. Но если нужно подключить внешний PostgreSQL, создайте `.env` рядом с `package.json`:

```env
DATABASE_URL=postgresql://postgres:postgres@127.0.0.1:5432/general
```

После этого можно создать таблицы командой:

```bash
npx drizzle-kit push
```

И запустить:

```bash
npm run dev
```

## Важное замечание

Если `.env` есть, приложение будет использовать базу из `DATABASE_URL`.

Если `.env` отсутствует, приложение автоматически использует встроенную базу PGlite.

## Диагностика внешнего PostgreSQL

Если всё-таки используете PostgreSQL/pgAdmin и хотите проверить подключение:

```bash
node scripts/check-db.js
```

## Сброс таблиц внешнего PostgreSQL

Если используете PostgreSQL через `.env` и нужно удалить только таблицы приложения:

```bash
node scripts/reset-local-db.js
npx drizzle-kit push
npm run dev
```

## Импорт вашего Excel-файла

1. Откройте приложение.
2. Перейдите в раздел **Реестр компаний** или нажмите на дашборде **Импортировать из Excel/CSV**.
3. Нажмите **Выбрать Excel / CSV файл**.
4. Выберите ваш `.xlsx` или `.xls` файл.
5. После чтения файла нажмите **Импортировать**.

Поддерживаются такие заголовки столбцов:

```text
ИНН
Наименование
Статус (экспортер, не экспортер, 2025 г.)
Статус работы ЦПЭ
Статус МСП
Отрасль
Основной вид деятельности
Продукция
Почта (Минпром)
Почта (ЦПЭ)
Телефон (Минпром)
Телефон (ЦПЭ)
Контактное лицо (Минпром)
Контактное лицо (ЦПЭ)
```

Поле **Сфера деятельности** приложение определяет автоматически по отрасли/продукции: АПК, Промышленность или Прочие.

## Основные функции

- Дашборд с KPI и аналитикой.
- Реестр компаний с поиском и фильтрами.
- Табличный и карточный вид.
- Карточка компании с вкладками: информация, контакты, экспорт, поддержка, CRM-задачи, аудит.
- Массовое редактирование.
- CSV импорт/экспорт.
- Динамические дополнительные поля.
- Роли: администратор, менеджер, аналитик.
- Светлая и тёмная тема.
